BioImage Suite Web Download Page.

This page contains desktop and commandline versions of the tools in BioImage Suite web. BioImage Suite Web also works directly in your browser and can be accessed from this webpage.

We gratefully acknowledge support from the NIH Brain Initiative under grant R24 MH114805 (Papademetris X. and Scheinost D. PIs).

  

Current Version

Version 1.0b4, Dec 10, 2018

  1. Desktop Applications
    • Microsoft Windows installer (~ 140 MB)
    • MacOS zipped .app file (~ 140 MB)
    • Linux (Ubuntu) zip file (~ 140 MB)

    The desktop applications are based on (and include) Electron.

  2. Command Line Tools
    • All-platform command line tools package. (~ 10 MB)
    • All-platform command line tools package with test data. (~ 50 MB)

    Please note that these distribution should work on both Windows, Mac OS and Ubuntu. In all three cases you will need to install Node.js v8.x or v10.x LTS.
